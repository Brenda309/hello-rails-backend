# Hello-rails-backend
This is an endpoint API 


## Built With

- Ruby On Rails (RoR)
- PostegreSQL



## Getting Started
To get a local copy up and running follow these simple example steps.


### Prerequisites
Ruby 3.1.2 or newer
Rails 7.0.4 or newer
Git
NPM
PostgreSQL
Modern web browser

### Setup
  $ git clone https://github.com/Brenda309/hello-rails-backend.git
  $ cd recipe_app/
### Install

     bundle install
     gem install rspec
### Usage

### Run project
    Execute command

    $ rails server

Then browse http://localhost:3000
